# Ensemble and Boosting Algorithms (Mini ML Projects)

This repository contains implementations of **Ensemble Learning and Boosting algorithms**, which are powerful techniques used to improve model performance by combining multiple models.

The projects focus on understanding how multiple weak learners can be combined to create a strong predictive model, along with practical implementations using real-world datasets.

---

## 📌 Repository Contents

This repository includes the following ensemble-based machine learning projects:

### 1. Decision Tree

Implementation of the **Decision Tree algorithm**, a fundamental supervised learning method used for classification and regression tasks.

Key concepts demonstrated:

* Tree structure (nodes, branches, leaves)
* Information Gain / Entropy
* Gini Index
* Recursive splitting
* Overfitting and pruning concepts

Dataset used:

* German Credit Risk dataset (`german_credit_risk.csv`)

---

### 2. Random Forest

Implementation of **Random Forest**, an ensemble method that combines multiple decision trees to improve accuracy and reduce overfitting.

Key concepts demonstrated:

* Bagging (Bootstrap Aggregation)
* Multiple decision trees
* Feature randomness
* Improved generalization
* Reduction of variance

Dataset used:

* HR Employee Attrition dataset

---

### 3. Boosting

Implementation of **Boosting techniques**, where models are trained sequentially and each new model focuses on correcting errors made by previous models.

Key concepts demonstrated:

* Sequential learning
* Error correction
* Weighted models
* Strong learner from weak learners
